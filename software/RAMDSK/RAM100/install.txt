Installation options

--- Install via TPDD ---

If you already have a TPDD client & server
then just use them to copy RAM100.CO to the Model 100/102
and then LOADM like any other .CO file.

  LOADM "RAM100.CO"
  note the TOP address in the error message
  CLEAR 0,<topaddr>

If you want to CALL the various functions from BASIC,
then repeat the LOADM after the CLEAR, or just execute
RAM100.CO from the main menu at least once.

--- Initial via BASIC loader ---

For Windows, https://github.com/bkw777/tsend

  C:> tsend.ps1 -file RAM100.DO

For Mac/Linux/Cygwin, https://github.com/bkw777/dl2

  $ dl -v -b RAM100.DO

--- Install from the RAMPAC itself after a cold reset ---

If you set this up, then you don't need a PC to reinstall any more.

Copy RAM100.CO to the RAMPAC.
RAM100.CO must be the first file on bank0.

Type-in the 4-line BASIC program RBOOT.100

You can use any bank instead of bank0 by making these changes:

Copy RAM100.CO to desired bank instead of bank0.
RAM100.CO must be the first file in the bank.

In RBOOT change all (2) occurances of OUT129 to OUT133, 137, or 141
